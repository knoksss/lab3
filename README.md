# Лабораторная работа номер 3 (алгоритмический пакет). Выполнила Китаева Дарья(М8О-102БВ-25)

## Работа с алгоритмическим пакетом:
Работа происходит аналогично, как с терминалом, для начала Вам даётся возможность выбрать ту функцию, которой Вы хотите воспользоваться(fib - для расчёта чисел Фибонначи, fact - для расчёта факториала и т.д.). Далее Вам на экран выводятся функции, которыми Вы можете воспользоваться в данном алгоритме.

Содержатся такие функции:
- Итеративное и рекурсивное вычисление чисел Фибоначчи
- Итеративное и рекурсивное вычисление Факториала
- Базовые сортировки(пузырьковая, быстрая, подсчетом, поразрядная, вёдерная, кучей)
- Структура данных stack (LIFO) с хранением минимального элемента за О(1) - константу
- Структура данных queue
- Генераторы тест-кейсов
- Бенчмаркинг (замер времени исполнения функций и формирование отсчета)

## Структура проекта
- В папке src находятся модули, отвечающие за реализацию алгоритмического пакета. Все функции распределены по папкам, а для их вызова используются функции, находящиеся в папку src. Главная функция, отвечающая за работу всей программы - main.py;
- В папке tests находятся тесты для всех функций;
- В файле Solved_practice.md приведены скриншоты и код решённой практики из лабораторной работы.
